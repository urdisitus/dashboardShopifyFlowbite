Reseteo de Contraseña

El campo "Usuario" es requerido.
El campo "Nueva Contraseña" es requerido.
El campo "Nueva Contraseña" no debe tener más de 100 caracteres.
El campo "Confirmar Contraseña" es requerido.
El campo "Confirmar Contraseña" no debe tener más de 100 caracteres.
El campo "Confirmar Nueva Contraseña" debe ser igual a la "Nueva contraseña"
